سياسة الخصوصية — تطبيق حالاتي (Halati)

آخر تحديث: يُحدَّث هذا التاريخ يدوياً مع كل تعديل لمحتوى هذه الصفحة.

تطبيق حالاتي ("التطبيق") من تطوير ياسر أبو علاء. توضح هذه السياسة ما يجمعه التطبيق من بيانات، وما لا يجمعه، وكيف تُستخدم.

1. البيانات التي لا يجمعها التطبيق

  • لا يجمع التطبيق اسمك، رقم هاتفك، أو أي معرّف شخصي
  • لا يرسل أي حالة واتساب أو ملف وسائط تحمّله إلى أي خادم خاص بنا لتخزينه أو تحليله
  • لا يبيع أو يشارك أي بيانات مع أطراف ثالثة لأغراض إعلانية
  • لا يحتوي التطبيق على إعلانات ولا على أدوات تتبع تسويقي

2. البيانات التي تبقى على جهازك فقط

تُخزَّن محلياً على جهازك فقط، ولا تُرسَل لأي مكان:

  • تفضيل اللغة والمظهر
  • رابط مجلد حالات واتساب الذي منحته صلاحية الوصول إليه
  • سجلّ صغير بأوقات نشر الحالات، يُستخدم فقط لحساب عدّاد الـ24 ساعة

4. الوسائط التي تُنزّلها من الإنترنت

يُرسَل الرابط الذي تلصقه فقط إلى خادم استخراج مخصص لاستخراج رابط الملف المباشر، ثم يُنزَّل مباشرة لجهازك. لا يُخزَّن الرابط أو الملف على الخادم بعد الاستخراج.

5. حذف بياناتك

احذف التطبيق، أو من إعدادات أندرويد: التطبيقات ← حالاتي ← التخزين ← مسح البيانات.

Privacy Policy — Halati App

Last updated: manually updated whenever this page's content changes.

Halati ("the App") is developed by Yasser Abu Alaa. This policy explains what data the App collects, what it doesn't, and how it's used.

1. Data the App does NOT collect

  • No name, phone number, or personal identifier is collected
  • No WhatsApp status or downloaded file is sent to any server we operate for storage or analysis
  • No data is sold or shared with third parties for advertising
  • No ads, no marketing/analytics tracking SDKs

2. Data kept only on your device

  • Language and theme preference
  • The WhatsApp status folder URI you granted access to
  • A small log of when viewed statuses were posted, used only for the 24h countdown

3. Permissions requested and why

PermissionPurpose
Media / StorageDisplay WhatsApp statuses and save downloaded videos/images
SAF folder pickerAccess WhatsApp's status folder on Android 11+
Install unknown appsInstall app updates (distributed outside Google Play)
InternetDownload links you paste, and check for updates
Note on Contacts permission: even if granted, the App cannot actually resolve a status owner's name — Android and WhatsApp provide no way to link a status file to a phone number. This is a general platform limitation, not specific to Halati.

4. Media you download from the internet

Only the URL you paste is sent to a dedicated extraction server to resolve a direct media link, which is then downloaded straight to your device. Neither the link nor the file is stored on that server afterward.

5. Deleting your data

Uninstall the App, or Android Settings → Apps → Halati → Storage → Clear data.
